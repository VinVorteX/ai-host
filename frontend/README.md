# Riva AI - React Frontend

Modern React frontend for Riva AI Assistant.

## Setup

```bash
# Install dependencies
npm install

# Development mode
npm run dev

# Build for production
npm run build
```

## Development

Frontend runs on: http://localhost:3000
Backend API: http://localhost:5000
